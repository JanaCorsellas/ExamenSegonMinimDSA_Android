En aquesta versió s'ha implementat una nova Activity que ens proporciona un petit formulari per solicitar i enviar peticions de millora pel joc ScapeRoom.
Les respostes del formulari serviran perquè els creadors del joc tinguin feedback i puguin millorar segons els gustos dels clients.


Canvis en el codi del projecte:
- nova Activity implementada (FormulariActivity)
- Layout implementat (activity_formulari)
- S'ha afegit una nova classe (Formulari)
- Ruta afegida a la interfície de la API
